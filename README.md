---

### ✨ Funcionalidades

- **Cuestionario sociométrico** — 15 preguntas configurables, 6 plantillas por nivel (Primaria, ESO, Bachillerato, Convivencia, Inclusión, General)
- **Grafo interactivo** — relaciones con vis-network: colores por rol, elección/rechazo, física suave
- **Métricas automáticas** — cohesión, densidad, aislamiento, reciprocidad
- **Predicciones** — detección de aislamiento, conflicto potencial, líderes positivos, baja cohesión
- **Editor de relaciones** — modo manual con matriz editable y drag & drop entre alumnos
- **Formación de equipos** — grupos equilibrados por rol, elecciones mutuas y rechazos
- **Distribución de clase** — plano visual con mesas, 3 disposiciones (cuadrícula, filas, en U), arrastra alumnos entre mesas
- **Exportación** — PNG, JSON, HTML, CSV, informes anonimizados para IA
- **Multi-idioma** — Español e Inglés

### 🔧 Tecnologías

Vue 3 · Tailwind CSS · vis-network · idb-keyval · html2canvas · Chart.js

Sin build, sin Node.js, sin servidor.
